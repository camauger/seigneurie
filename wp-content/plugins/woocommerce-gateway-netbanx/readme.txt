=== WooCommerce Paysafe Gateway ===

This plugin allows you to use Paysafe payment gateway within your WooCommerce store.

== Description ==

A plugin that allows you to extend your WooCommerce installation with the Paysafe payment processor

This plugin was created by: vanbo.develops(Ivan Andreev)

== Installation ==

1. Upload the `woocommerce-gateway-netbanx` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to `WooCommerce > Settings > Payment Gateways` to configure the Paysafe gateway settings.
